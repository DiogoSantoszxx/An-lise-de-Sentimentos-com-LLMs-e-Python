# An-lise-de-Sentimentos-com-LLMs-e-Python
O fluxo do projeto envolve:  Leitura de avaliações a partir de um arquivo .txt  Processamento de cada resenha com um modelo de linguagem  Retorno estruturado em JSON  Classificação automática de sentimentos (positivo, negativo e neutro)  Consolidação dos resultados com contadores e organização dos dados
